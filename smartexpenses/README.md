SmartExpenses ( expesnses tracker) - python

# Description
smartexpenses is a command-line expense tracking system built using python. it allows users to record daily expenses, analyze spending patterns and view summaries. 

this project shows file handling, data processing and basic system design. 

feature:
Add expenses
view all expenses
calculate total spending 
categorywise summary
monthly expense analysis
data stored usinh CSV file

Technologies used:
python
CSV module
Datetime module
OS module

Structure:
main.py --> main program file
expenses.csv --> stores expense data
README.md --> documentation
requirements.txt --> dependencies (not required for now)

How to run:
1.Install python
2.Open project folder in VS code
3.Run: Python main.py
4.Use menu options to interact with the system
